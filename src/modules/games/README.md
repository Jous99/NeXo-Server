# Módulos de juego

Esta carpeta contiene los módulos de servidor para juegos específicos que requieren infraestructura propia (salas, matchmaking, relay de datos, etc.).

## Cómo añadir un módulo

Ver [`docs/modules.md`](../../../../docs/modules.md) para la guía completa.

## Módulos planificados

- `matchmaking/` — Sistema genérico de salas y emparejamiento
- `mario-kart/` — Servidor para Mario Kart
- `splatoon/` — Servidor para Splatoon
