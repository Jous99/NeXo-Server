#  CÓMO FUNCIONA (importante entender esto):

#  Este archivo NO te conecta a Nintendo — hace exactamente lo contrario.
#  Atmosphere intercepta el DNS de la Switch. Cuando un juego o el sistema
#  intenta resolver "dauth-lp1.ndas.srv.nintendo.net", Atmosphere devuelve
#  la IP de TU SERVIDOR en lugar de la IP real de Nintendo.
#
#  Nintendo nunca ve tu consola → No puede detectar el CFW → Sin ban.
#
#  Es el mismo principio que usa 90DNS (bloqueo de servidores Nintendo),
#  pero en lugar de bloquear, redirigimos a nuestro servidor que responde
#  con stubs válidos para que los juegos funcionen.
#
#  INSTRUCCIONES:
#  1. Reemplaza TU_IP_AQUI por la IP pública de tu servidor NeXoNetwork
#  2. Copia este archivo a la SD: /atmosphere/hosts/default.txt
#  3. Reinicia la Switch con Atmosphere
#
#  RECOMENDACIÓN EXTRA: Usa emuMMC (NAND emulada en la SD) en lugar de
#  sysMMC para el CFW. Así tu Switch real nunca se expone aunque algo falle.
# ══════════════════════════════════════════════════════════════════════════════
