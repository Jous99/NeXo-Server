#  NeXoNetwork — Archivo de hosts para Atmosphere CFW
#  Dominio: nexonetwork.space
#
#  CÓMO INSTALAR:
#    1. Copia este archivo a la tarjeta SD de tu Switch:
#       /atmosphere/hosts/default.txt
#    2. Reinicia la Switch con Atmosphere
#
#  CÓMO FUNCIONA (importante):
#    Este archivo NO conecta a Nintendo — hace exactamente lo contrario.
#    Atmosphere intercepta el DNS internamente. Cuando la Switch intenta
#    resolver "dauth-lp1.ndas.srv.nintendo.net", Atmosphere devuelve
#    la IP de nexonetwork.space en su lugar.
#    Nintendo nunca ve tu consola → sin detección de CFW → sin ban.
#    Es el mismo principio que 90DNS, pero en vez de bloquear, redirige
#    a tu servidor que responde con datos válidos.
#
#  NOTA SOBRE EL FORMATO:
#    Atmosphere soporta dominios además de IPs en este archivo.
#    Si algo no funciona, reemplaza "nexonetwork.space" por la IP
#    directa del servidor (ej: 185.123.45.67).
#    Puedes encontrar la IP con: ping nexonetwork.space
# ══════════════════════════════════════════════════════════════════════════════
